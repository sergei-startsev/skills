# Agent Skills

A collection of custom [agent skills](http://agentskills.io/).

## Marketplace Installation

```
/plugin marketplace add sergei-startsev/skills
```

## Skills

### [create-announcement](./skills/create-announcement)

Creates announcements for IM platforms (MS Teams, Slack) based on Pull Request(s).

```
/plugin install mgmt@sergei-startsev-skills
```
